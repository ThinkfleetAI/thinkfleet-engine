# `thinkfleetbot setup`

Initialize `~/.thinkfleetbot/thinkfleetbot.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
thinkfleetbot setup
thinkfleetbot setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
thinkfleetbot setup --wizard
```
