# `thinkfleetbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
thinkfleetbot pairing list whatsapp
thinkfleetbot pairing approve whatsapp <code> --notify
```
